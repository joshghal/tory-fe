# TORY

## Description
TORY is a web app built with Next.js that helps users explore and understand token data. It connects to platforms like Arkham Intelligence, Token Terminal, LunarCrush, and CryptoRank to pull in relevant financial and on-chain data.

The name "Tory" comes from combining "Token" and "Inquiry" — because that's exactly what it does: help you inquire about tokens.

Tory also provides AI-generated insights that analyze:
- Token unlock schedules
- Token financial statements

These insights are structured to highlight bullish or bearish signals based on measurable, quantitative trends.

## Installation
To get started:

```bash
git clone https://github.com/chaleoncompaleon/tory-fe
cd tory-fe
yarn
```

## Usage
To run the app locally:

```bash
yarn dev
```

Open your browser to `http://localhost:3000`.
